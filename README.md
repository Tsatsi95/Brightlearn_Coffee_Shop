Project Title: Bright Coffee Shop Sales Analysis

1. Project Overview
This project involves a comprehensive sales analysis for Bright Coffee Shop to assist a new CEO in driving revenue growth and improving product performance. As a Junior Data Analyst, I extracted actionable insights from historical daily transactional data to identify high-performing products, peak sales periods, and growth opportunities

2. Objectives
The primary goal was to provide data-driven answers to key business questions:
- Revenue Leaders: Identifying which products generate the most income.
- Temporal Performance: Determining the most profitable times of day using 30-minute intervals.
- Trend Analysis: Mapping sales trends across various product categories and timeframes.
- Strategic Recommendations: Providing actionable advice to improve overall sales performance

3. Tech Stack & Tools
- Planning: Luci chart (Architecture & Data Flow Diagrams).
- Processed Dataset: Cleaned transactional data with new calculated fields, including:
  * total_amount: Calculated as $unit\_price \times transaction\_qty$.
  * transaction_time_bucket: Grouped into 30-minute intervals for granular time analysis.
- SQL Scripts: Code used for data casting (e.g., fixing decimal formatting), transformations, and grouping.
- Interactive Dashboard: Visual representations of revenue per product type, peak sales intervals, and volume by category using Excel.
- Executive Presentation: A summary of methodology, key insights, and strategic recommendations such as targeted marketing for slow periods and loyalty program implementation using PowerPoint.
